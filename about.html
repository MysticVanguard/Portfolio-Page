Growth

The first project I ever worked on in my computer science career is the bot I created called AquaMarine (The first version of the code looking like this). I had learned the very basics of Python using a CodeCademy Course and had multiple friends who knew Python and created bots, so I used Aqua to put my knowledge to the test and learn the intermediate skills when coding in Python. The bot's first version was created over the 2021 summer and was ready to leave testing and be public by August, where it quickly grew to hit the 100 server mark in February of 2022. Since then it became verified and sits at over 350 servers one year later, February of 2023, and has had 19 major updates to the code.
